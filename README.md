# 学生信息管理系统(基于JavaSE)

## 一、实现功能

- 授予管理员登录与注册功能
- 授予学生登录与注册功能
- 管理员可以对学生信息进行CRUD
- 学生只能进行查询信息，没有权限修改信息

## 二、特色功能

- 注册管理员密码时使用的时正则表达式
- 随机给定4位验证码进行登录判断
- 可以跳级进行查询与操作管理系统
